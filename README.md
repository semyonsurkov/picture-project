## Данный репозиторий позволяет скопировать готовое и настроенное окружение для проекта "Pictures"

## Set - up
1. Install the following packages:
```
npm install -g yarn
```
2. Install dependencies:
```
yarn
```

## How to run the project:
```
yarn dev
```
## How to run the project:
```
yarn dev
```
## How to build the project:
```
yarn build
```
